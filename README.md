# Pymongo-X-TKInter
Programa feito com fins acadêmicos onde a partir do Python realizo conexões com o banco de dados MongoDB. Nele, é possivel adicionar dados em uma tabela (alguns, criptografados), listar todos os dados com a criptografia e, caso o usuário possua a chave de acesso, pode listar todos os dados descriptografados

# COMO RODAR
1. Mude os dados da conexão com o banco para os dados do seu cluster  (linha 32)
2. Mude o caminho DATA_DIR (linha 10) para algum de preferencia (um arquivo texto com sua key sera salvo nesse diretório, caso seja sua primeira vez rodando o programa, para que voce possa acessar os dados descriptografados)
3. Rode o programa e aguarde (pode demorar um pouco)
